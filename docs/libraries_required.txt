PROJECT DEPENDENCIES & LIBRARIES
================================

This document lists all necessary libraries and tools required to run the CRM project on a new machine.

PREREQUISITES
-------------
1. Python 3.10+
2. Node.js 18+ & npm
3. MySQL Server 8.0+

BACKEND LIBRARIES (Python)
--------------------------
Install these using pip:
pip install fastapi uvicorn mysql-connector-python python-dotenv pydantic requests

Specific Libraries:
- fastapi: Web framework for building APIs.
- uvicorn: ASGI server implementation, using uvloop and httptools.
- mysql-connector-python: Official MySQL driver.
- python-dotenv: Reads key-value pairs from a .env file and can set them as environment variables.
- pydantic: Data validation using Python type hints.
- requests: Simple HTTP library for Python (used for some internal calls or tests).

FRONTEND LIBRARIES (JavaScript/React)
-------------------------------------
Install dependencies using npm in the 'frontend' directory:
npm install

Core Dependencies:
- react: ^18.x
- react-dom: ^18.x
- vite: Build tool and dev server.
- axios: Promise based HTTP client for the browser.
- lucide-react: Beautiful & consistent icon toolkit.
- prop-types: Runtime type checking for React props.

SETUP INSTRUCTIONS
------------------
1. Database: Import 'CRM.sql' (or latest schema) into your MySQL server.
2. Backend:
   - Create a .env file in 'backend/' with DB credentials (DB_HOST, DB_USER, DB_PASSWORD, DB_NAME).
   - Run: python api_server.py
3. Frontend:
   - Navigate to 'frontend/'.
   - Run: npm install
   - Run: npm run dev

PROJECT STRUCTURE
-----------------
/CRM (Root)
  /backend  - Python FastAPI Server
  /frontend - React Vite Application
  /docs     - Documentation and Manuals
